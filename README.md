# PowerBI-Sales-Dashboard
This project presents an interactive Sales and Revenue Dashboard developed using Power BI. The dashboard analyzes product performance, regional sales distribution, revenue trends, and key business insights through data visualization techniques.

## Tools Used
- Power BI
- Microsoft Excel

## Dashboard Features
- KPI Cards for Total Sales and Revenue
- Product-wise Sales Analysis
- Revenue Distribution by Region
- Sales Trend Visualization
- Interactive Charts and Visuals

## Purpose
The purpose of this project is to demonstrate data analysis, dashboard design, and visualization skills using Power BI.
